# Árbol 3D Navideño
![Arbol 3D navideño](https://github.com/CristhianE-GuinoQ/CristhianE-GuinoQ.github.io/blob/ad09e8b1a6b2cf675b4cc2a948dc5a0da5e92072/arbol3d.png)

Este proyecto muestra un árbol de Navidad en 3D con hologramas y animaciones interactivas en el navegador.

## Funcionalidades
- Árbol 3D con base holográfica.
- Texto “FELIZ NAVIDAD” animado.
- Papá Noel animado moviéndose alrededor del árbol.
- Efectos holográficos en adornos y base.
- Protección de líneas de terminal: si se borran o modifican las 4 líneas principales, aparece pantalla negra.

## Uso
1. Abrir `index.html` en un navegador moderno.
2. Disfrutar de la animación y los efectos holográficos.

## Tecnologías
- HTML, CSS, JS
- Three.js
